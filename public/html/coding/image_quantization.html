Image Color Quantization with Window Averaging

Jump to the interactive area!

Image quantization is a simple lossy compression technique in which a range colors in a given image is reduced to a smaller range of colors. Useful for formatting images for certain displays and in image compression. I find it particularly useful in severely limiting colors in images that I would like to make a print out of. Along with the window averaging functionality, I've found that the image quantizer creates a great simplified image.


I came across Photopea's image quantizer when looking for a way to simplify an image and figured I would create my own image quantizer as coding a challenge. Along with Tkinter, I was able to make a simple GUI based application that allowed me to quantize images without all the other fully fledged features of a dedicated image editor. This also allowed me add in a simple window averaging function that lends itself to rounding out and simplifying harsh edges in images.
